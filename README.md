### 🧺 センタクびより – Weather × Life App (新作)

「今日は洗たくできるかな？」  
そんな日常の小さな悩みをテクノロジーで解決するWebアプリです。  

OpenWeather API の3時間ごとの天気データをもとに、  
**気温・湿度・風速・降水確率**から「乾きスコア」を自動計算。  
最も洗たく物が乾きやすい時間帯を提案します。  

🧮 Dry Score = 2 × 気温 + 3 × 風速 − 0.8 × 湿度 − 0.5 × 降水確率  

- UIは固定ヘッダーとグリッドレイアウトで、スマホでも見やすく設計  
- デモ用APIキーを使用し、実際の天気データを反映  

👉 [センタクびよりを見る](https://mei-oku.github.io/sentaku-biyori/)

![センタクびより スクリーンショット](https://github.com/MEI-OKU/sentaku-biyori/blob/main/screenshot.png?raw=true)

### 🏔️ Local Connection App
札幌と地方をつなぐ、**地域創生×IT** をテーマにしたマップ型Webアプリです。  
地域の空き家・コワーキングスペース・活動拠点などを可視化し、  
都市と地方をゆるやかにつなぐ「関係人口創出」の仕組みを目指しています。

- OpenWeather API + Leafletで、地図と天気情報をリアルタイム表示  
- 地域データはJSONで管理し、将来的に動的データベースへ拡張予定  
- UIはFigma構想をもとに、シンプルかつ直感的な設計に  

🗺️ **[Local Connection Appを見る](https://mei-oku.github.io/local-connection-app/)**

![Local Connection App Screenshot](https://github.com/MEI-OKU/local-connection-app/blob/main/images/screenshot.png)
